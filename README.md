# cheeth-ps
<h3>Gerenciador de Automação com Foco no Adobe Photoshop</h3>
<p>
  Este projeto possibilita que designers usuários do Photoshop criem e gerenciem pilhas de automações com apenas alguns cliques.  
  Imagine que você tem um card com um layout padrão, mas as informações precisam ser atualizadas constantemente.  
  Em vez de fazer isso manualmente, você pode criar uma automação usando o cheeth-ps. O processo é realizado com máxima velocidade e eficiência.
</p>
<br>
<h4>Como usar o projeto?</h4>
<ol>
  <li>Instale o Python</li>
  <li>Instale as dependências listadas no arquivo <code>requirements.txt</code></li>
  <li>Configure as pastas dos seus arquivos:
    <ul>
      <li>Crie uma pasta contendo o arquivo PSD</li>
      <li>Crie outra pasta para armazenar as imagens geradas pelo App</li>
    </ul>
  </li>
  <li>Desfrute e use à vontade!</li>
</ol>
<p>
  <strong>Observação:</strong> Este projeto está em versão beta. Algumas funcionalidades estão ausentes e podem haver bugs. Ele será atualizado com o tempo, mas, no momento, é isso.
</p>
